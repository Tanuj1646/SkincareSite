# Blush Skincare Website

A minimal, responsive skincare brand portfolio and shop built with Flask, HTML, CSS, and JavaScript.

## Features
- Home, Shop, Product Detail, About, Contact, and Cart pages
- Product catalog stored in Python data structures
- Cart managed in Flask session
- Contact form with flash messages
- Responsive mobile-friendly layout
- Pastel, elegant skincare design

## Setup
1. Install Flask
```bash
pip install flask
```
2. Run the app
```bash
python app.py
```
3. Open `http://127.0.0.1:5000` in your browser.

## Project structure
- `app.py` — Flask application and routes
- `templates/` — Jinja2 HTML templates
- `static/css/style.css` — styling
- `static/js/main.js` — mobile menu behavior
- `static/images/` — product and hero illustrations
